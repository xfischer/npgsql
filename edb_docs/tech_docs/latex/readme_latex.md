# LaTeX protocol datagrams

Those diagrams were generated using [epascap2latex](https://github.com/EnterpriseDB/epascap2latex) (a superset of [pgcap2latex](https://github.com/EnterpriseDB/pgcap2latex))

## Installation

- VSCode
- Add [LaTeX Workshop Extension](https://marketplace.visualstudio.com/items?itemName=James-Yu.latex-workshop) extension


## Usage

Add a new .tex file and start editing!
PDF Files are generated on save.
