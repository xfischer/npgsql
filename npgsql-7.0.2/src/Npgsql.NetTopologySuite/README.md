EDB .NET Connector is the .NET data provider for EDB Postgres Advanced Server. It allows you to connect and interact with EDB Postgres Advanced Server server using .NET.

For more information, [see the EDB documentation](https://www.enterprisedb.com/docs/net_connector/latest/) and [the community NetTopologySuite plugin documentation page](https://www.npgsql.org/doc/types/nts.html).
